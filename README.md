pandas-util
===


# Requirements

* (optional) pyenv  
  https://github.com/pyenv/pyenv
* python 3
  * pandas  
    https://pandas.pydata.org/

# Prerequisites

 * (optional) pyenv
    ```
    $ pyenv install 3.9.13
    $ pyenv local 3.9.13
    ```
 * (optional) venv
    ```
    $ python -m venv pandas-util
    $ source pandas-util/bin/activate
    ```
 * pip
    ```
    $ python -m pip install -r requirements.txt
    ```
